  1. Create `python/src/odoo_gen_utils/validation/docker_runner.py`:

     - `check_docker_available() -> bool`: Use `shutil.which("docker")` to check CLI presence. Also try `subprocess.run(["docker", "info"], capture_output=True, timeout=10)` and check returncode == 0. Return False if either check fails.

     - `get_compose_file() -> Path`: Return `Path(__file__).parent.parent.parent.parent.parent / "docker" / "docker-compose.yml"` (navigate from validation/ up to project root). This resolves to the extension's docker/ directory.

     - `_run_compose(compose_file: Path, args: list[str], env: dict[str, str], timeout: int = 120) -> subprocess.CompletedProcess`: Helper to run `docker compose -f {compose_file} {args}` with environment variables and timeout. Merges env with os.environ.

     - `_teardown(compose_file: Path, env: dict[str, str]) -> None`: Run `docker compose -f {compose_file} down -v --remove-orphans` to tear down containers and volumes. Catches all exceptions (teardown must never raise).

     - `docker_install_module(module_path: Path, compose_file: Path | None = None, timeout: int = 300) -> InstallResult`:
       - If not check_docker_available(): return InstallResult(success=False, log_output="", error_message="Docker not available")
       - If compose_file is None: use get_compose_file()
       - Set env: MODULE_PATH=str(module_path.resolve()), MODULE_NAME=module_path.name
       - try:
         - `_run_compose(compose_file, ["up", "-d", "--wait"], env, timeout=120)` -- start services
         - `result = _run_compose(compose_file, ["exec", "-T", "odoo", "odoo", "-i", module_path.name, "-d", "test_db", "--stop-after-init", "--no-http", "--log-level=info"], env, timeout)` -- install module
         - Parse result.stdout + result.stderr with parse_install_log
         - Return InstallResult(success=parsed_success, log_output=combined_output, error_message=parsed_error)
       - except TimeoutExpired: Return InstallResult(success=False, ..., error_message="Timeout")
       - finally: _teardown(compose_file, env)

     - `docker_run_tests(module_path: Path, compose_file: Path | None = None, timeout: int = 600) -> tuple[TestResult, ...]`:
       - If not check_docker_available(): return ()
       - Same pattern as install but command includes `--test-enable` and `--log-level=test`
       - Parse output with parse_test_log
       - Always teardown in finally
