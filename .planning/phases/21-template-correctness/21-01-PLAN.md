5. **Build inherit_list** as before (explicit_inherit first), then only inject mail mixins if:
   - `chatter` is truthy AND
   - `"mail"` is in `spec.get("depends", [])` AND
   - parent model (explicit_inherit) is NOT another model in the same module (to avoid double mail.thread)

6. **Keep existing behavior intact**: All existing tests must still pass. The only change is that line items and explicitly-opted-out models skip injection.

Do NOT modify any template files. The fix is entirely in `renderer.py` -- templates already render whatever `inherit_list` contains.
