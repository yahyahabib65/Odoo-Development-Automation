       - Given a source string, an ast.Call node, and the index of a keyword to remove, splice out the keyword text using AST positions (kw.lineno, kw.col_offset, kw.end_lineno, kw.end_col_offset).
       - Handle comma cleanup: if keyword was last, remove preceding comma. If first, remove trailing comma. If middle, remove trailing comma.
       - Handle blank line cleanup: if removing the keyword leaves an empty line, remove that line.
       - Handle the case where the entire line contains only the keyword + comma + whitespace.
       - CRITICAL: AST positions are 1-based for lineno, 0-based for col_offset.

    2. `_splice_remove_dict_entry(source: str, key_node: ast.expr, val_node: ast.expr) -> str`:
       - Remove a key-value pair from a dict literal using AST positions.
       - Span: from key_node.lineno to val_node.end_lineno (inclusive).
       - Remove entire lines in that range. Handle trailing comma on the line before or after.

    3. `_splice_rename_keyword(source: str, kw: ast.keyword, new_name: str) -> str`:
       - Replace the keyword argument name at its precise position (kw.lineno, kw.col_offset) without touching anything else.
       - The keyword name spans from col_offset to col_offset + len(kw.arg).

    4. `_find_call_at_line(tree: ast.Module, target_line: int) -> ast.Call | None`:
       - Walk the AST to find an ast.Call node whose line range includes target_line.
