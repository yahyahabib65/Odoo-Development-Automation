    2. Create `src/odoo_gen/ui/display.py`:
       - `announce_stage(stage_name: str) -> None` -- prints styled stage announcement (e.g., `[bold green]>>> {stage_name}[/]`).
       - `show_module_summary(module_name: str, output_path: str, files_created: list[str]) -> None` -- renders a Rich Panel showing what was created (module name, path, file list).
       - `show_config_table(config: dict) -> None` -- renders a Rich Table with Key/Value columns showing current config.
       - `show_error(message: str) -> None` -- prints `[bold red]Error:[/] {message}`.
       - `show_stub_message(command: str) -> None` -- prints a styled message like "[dim]{command} will be available in a future version.[/]".
       - All functions use `get_console()` so quiet mode is respected.

    3. Create `src/odoo_gen/ui/__init__.py` exporting all display functions and console accessors.

    4. Create `src/odoo_gen/state/models.py`:
       - `PipelineState(BaseModel)` with `model_config = {"frozen": True}`.
       - Fields: `module_name: str`, `module_technical_name: str`, `odoo_version: str = "17.0"`, `edition: str = "ce"`, `output_dir: str`, `stage: str = "scaffold"`, `created_at: str` (default factory: datetime.now().isoformat()), `updated_at: str` (same).
       - Method `with_stage(new_stage: str) -> PipelineState` that returns `self.model_copy(update={"stage": new_stage, "updated_at": datetime.now().isoformat()})`.

    5. Create `src/odoo_gen/state/persistence.py`:
       - `STATE_DIR_NAME = ".odoo-gen"` constant.
       - `save_state(state: PipelineState, base_dir: Path) -> Path` -- writes state as JSON to `base_dir / module_technical_name / STATE_DIR_NAME / state.json`. Creates directories if needed. Returns the path written.
       - `load_state(module_dir: Path) -> PipelineState | None` -- reads from `module_dir / STATE_DIR_NAME / state.json`. Returns None if file does not exist. Handles invalid JSON gracefully (returns None, logs warning).

    6. Create `src/odoo_gen/state/__init__.py` exporting `PipelineState`, `save_state`, `load_state`.

    7. Create `tests/unit/test_display.py`:
