### Render Pipeline (10/10 stages wired)

All stages return `Result[list[Path]]` and are wired into `render_module()`.

### Critical Integration Bug

**Archival wizard + state field model** — `view_form.xml.j2` line 20 accesses `wizard.trigger_state` unconditionally. Archival wizards (Phase 34) don't have `trigger_state`. When a model has both `archival: true` and a `state` Selection field, `render_models()` crashes with StrictUndefined error, aborting the entire pipeline.

**Fix:** Guard with `wizard.get('trigger_state')` or add default to archival wizard dict.

### Override Flag Merge (verified)

Phase 29 constraints + Phase 34 bulk/cache correctly OR the `has_create_override`/`has_write_override` flags. Single create/write method renders combined logic.
